======================================================================
Connecting to Star Wars MUSH is possible through raw Telnet, though doing so is not recommended. Instead, it is suggested that you browse the following list of clients, and select the one that fits your play-style and tastes.

{| class="wikitable"
! colspan="2" | OS Independent
|-
|[http://potatomushclient.com/ Potato]
|Free. Runs on Windows XP, Vista, 7, and 8, as well as Linux, and Mac OS X. Raspberry Pi functionality is in the works. For help with client, see Jaspar in-game. For a list of features, look [https://github.com/potatomushclient/potato/wiki here] .
|-
|[http://www.ggsoft.org/ggmud/ GGMud]
|Free. Java-based.
|-
|[http://netherland.ath.cx/nether/content.php?topic=thebean Java Bean MUD Client]
|Free. Java-based. No longer maintained.
|-
|[http://lyntin.sourceforge.net/ Lyntin]
|Free. Python-based.
|-
|[http://tinyfugue.sourceforge.net/ tinyfugue]
|Free, but not recently maintained (c. 2007). Not for novice computer users due to command-line interface, and lack of GUI.
|-
|[http://tintin.sourceforge.net/ TinTin++]
|Free. Java-based. Runs on any OS (OS X, Windows, UNIX). Requires base program; not for novice computer users.
|-
|[http://www.mudconnect.com/fmud/ TMC Flash Client]
|Free, browser-based.

Do people mind if we modify this template to replace the 

<nowiki>[[Talk:{{PAGENAME}}|talk page]]</nowiki>

markup with something like the following markup (not sure if this will exactly work as quoted here)

<nowiki>[{{SERVER}}{{localurl:{{NAMESPACE}}:{{PAGENAME}}|action=edit}} talk page]</nowiki>

...as is done on the stub templates? It would clean out Talk:<blah> items from the Wanted Pages list. Right now, whenever one applies the Wookieepedia template to a Wanted Page, the Talk: redlink '''replaces''' the old redlink, which doesn't really help in cleaning out the list.

The downside to this method is that it would point to the messy edit versions of the talk pages, rather than to the talk pages themselves.

The same change could be made to the Dispute, Dispute pov, Expand, Expand Alt, and Purpose templates. --[[User:Lolkje|Lolkje]] 17:51, 13 April 2007 (UTC)

* Done.  Feel free to apply this snippet to other templates.  It takes care of the edit and points to the '''talk''' page.<pre>[{{SERVER}}{{localurl:{{TALKPAGENAME}}}} talk page]</pre> -- [[User:Xerxes|Xerxes]] 18:13, 13 April 2007 (UTC)
